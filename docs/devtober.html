Devtober 2020

Wooo! It's October! Other than being one of my favorite months, I'm excited for October this year because I'm participating in a game jam called Devtober. It's a challenge to work on a game a bit every day for the month of October, and if you want to learn more (or participate in it) I'd recommend checking out their page. Other responsibilities kicked my butt today, so I'm just going to write this overview of the idea and maybe a toot.

I'll be updating this page with dev-type news of my progress, as well as my mastodon account (with the hashtag `#devtober`).

My basic idea is an open world multiplayer where the players never see each other, only the abstract structures they can create. I'll be mostly brainstorming that idea for the rest of this week, but for now I need to sleep!
