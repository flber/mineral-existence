Plugin Documentation

This is an experimental page to document the plugins that run to modify small parts of certain pages on this site. The documentation for each plugin is generated by pulling comments from the code (so it only works on scripts which are not compiled).

doc-gen

  1. goes through all files in plugins directory, sorting by last edit time
  2. reads plugin for lines containing "#DOC: " and adds them to `plugin_docs` string
  3. adds `plugin_docs` string to `docs`

tree

  1. not currently in use

title

  1. finds first h2 and uses text inside as title

stats

  1. starts with four main pages
  2. traverses through all pages found so far looking for new links
  3. finds all links
  4. eliminated links which are external
  5. adds those links to the found list if they're not already there
  6. adds abspath to each page string
  7. removes found links from all pages
  8. adds orphaned pages' names and links to list

music

  1. gets music path
  2. gets album names and adds them to list

latest

  1. gets granite path
  2. replaces {{latest}}
  3. sorts files in /pages by reversed edit time
  4. finds title
  5. adds edit time to list
  6. adds file link and title to list

files

  1. uses some cursed recursize walk function
  2. just removes base file path
  3. adds file name and link to list

feed

  1. base xml string
  2. adds last run date to xml string
  3. closes xml string
  4. writes to xml file

date

  1. replaces "{{date}}" with the date.
